Atmo

Sensor Information

Atmo will be able to give the user access to relevant information necessary for camping. The design can detect the current temperature and humidity using the DHT11 sensor. Air quality will also be measured using the SPG30 sensor. This sensor detects a wide range of volatile organic compounds (VOCs) as well as H2. The Bluetooth module, ZS-040, is also integrated into the design. This is what allows for there to be connections between the Arduino and the website.
